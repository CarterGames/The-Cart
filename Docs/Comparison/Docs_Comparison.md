### Numerical
Provides an enum for common math comparisons:

| Original            | Enum `inspector label`          | Enum `code label` |
|:-------------|:-----------|:-----------|
| <    |  < (Less Than)        | LessThan |
| <= | <= (Less Than Or Equal To)      | LessThanOrEqual |
| = | = (Equal To)      | Equals |
| >= | >= (Greater Than Or Equal To)      | GreaterThanOrEqual |
| > | > (Greater Than)      | GreaterThan |
